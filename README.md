Personal AI Assistant (Modular & Local)

Программный комплекс персонального ассистента с модульной архитектурой, работающий на базе локальной языковой модели Llama 3. Проект ориентирован на обеспечение приватности данных и автономность управления ПК.

Основные особенности:
1) Локальный ИИ: Использование модели Llama 3 гарантирует работу без передачи данных на внешние сервера.
2) Модульная архитектура: Система разделена на ядро (bot_core) и независимые плагины. Новые функции добавляются без изменения основного кода.
3) Безопасность: Автоматическое шифрование истории чатов по алгоритму AES-256 (CBC).
4) Гибридное управление: Поддержка текстовых команд и голосового ввода (Google STT).

Технологический стек:
1) Language: Python 3.10+
2) AI Core: Meta Llama 3 (через llama-cpp-python / ollama)
3) Security: PyCryptodome (AES-256)
4) API Integrations: Google Speech-to-Text, OpenWeatherMap

Структура проекта:

/core — логика ядра ассистента и PluginManager;

/plugins — папка с функциональными модулями;

encryption_plugin.py — защита данных;

weather.py — прогноз погоды;

translator.py — двусторонний переводчик;

program_launcher.py — запуск приложений (Word, калькулятор и др.);

base_plugin.py — абстрактный класс для разработки новых модулей;

Безопасность:

Проект реализует локальное хранение данных. Файл истории chat_history.json шифруется в chats.encrypted. Ключ доступа генерируется индивидуально и хранится локально в secret.key.
